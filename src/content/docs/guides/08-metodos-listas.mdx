---
title: Métodos de listas
---
Las listas en Python vienen con una serie de métodos incorporados que nos permiten realizar operaciones sobre ellas de manera eficiente.

Aquí describimos algunos de los métodos más utilizados.

1️⃣ len()

2️⃣ max()

3️⃣ min()

4️⃣ sorted() y sort()

5️⃣ copy()

6️⃣ count()

7️⃣ reverse()

## 1️⃣ len()
---
Devuelve la longitud de la lista, es decir, el número de elementos que contiene.

```python
mi_lista = [1, 2, 3, 4]
print(len(mi_lista))  # Salida: 4
```

## 2️⃣ max()
---
Devuelve el valor máximo de la lista. Los elementos deben ser comparables entre sí.

```python
mi_lista = [1, 2, 3, 4]
print(max(mi_lista))  # Salida: 4
```

## 3️⃣ min()
---
Devuelve el valor mínimo de la lista.

```python
mi_lista = [1, 2, 3, 4]
print(min(mi_lista))  # Salida: 1 
```

## 4️⃣ sorted() y sort()
---
- sorted(): Devuelve una nueva lista ordenada, sin modificar la original.

- sort(): Ordena los elementos de la lista en orden ascendente. Modifica la lista original

```python
mi_lista = [3, 1, 4, 2]
print(sorted(mi_lista))  # Salida: [1, 2, 3, 4]
print(mi_lista)  # La lista original permanece sin cambios: [3, 1, 4, 2]

mi_lista.sort()
print(mi_lista)  # Salida: [1, 2, 3, 4]
```

## 5️⃣ copy()
---
Devuelve una copia superficial de la lista

```python
mi_lista = [1, 2, 3]
copia_lista = mi_lista.copy()
print(copia_lista)  # Salida: [1, 2, 3]
```

## 6️⃣ count()
---
Devuelve el número de veces que aparece un elemento en la lista.

```python
mi_lista = [1, 2, 2, 3]
print(mi_lista.count(2))  # Salida: 2
```

## 7️⃣ reverse()
---
Invierte el orden de los elementos en la lista, modificando la lista original.

```python
mi_lista = [1, 2, 3]
mi_lista.reverse()
print(mi_lista)  # Salida: [3, 2, 1]
```

## Métodos de adición en listas
---
1️⃣ append()

2️⃣ extend()

3️⃣ insert()

## 1️⃣ append()
---
Agrega un elemento al final de la lista.

```python
mi_lista = [1, 2, 3]
mi_lista.append(4)
print(mi_lista)  # Salida: [1, 2, 3, 4]
```

## 2️⃣ extend()
---
Agrega todos los elementos de una lista iterable (otra lista, tupla, etc.) al final de la lista actual.

```python
mi_lista = [1, 2]
mi_lista.extend([3, 4])
print(mi_lista)  # Salida: [1, 2, 3, 4]
```

## 3️⃣ insert()
---
Inserta un elemento en una posición específica.

```python
mi_lista = [1, 3, 4]
mi_lista.insert(1, 2)
print(mi_lista)  # Salida: [1, 2, 3, 4]
```

## Métodos de eliminación en listas
---
1️⃣ pop()

2️⃣ remove()

3️⃣ clear()

## 1️⃣ pop()
---
Elimina y devuelve el elemento en la posición especificada. Si no se proporciona un índice, elimina el último elemento.

```python
mi_lista = [1, 2, 3]
ultimo_elemento = mi_lista.pop()
print(ultimo_elemento)  # Salida: 3
print(mi_lista)  # Salida: [1, 2]
```

## 2️⃣ remove()
---
Elimina la primera aparición de un elemento específico en la lista. Si el elemento no está presente, se genera un error

```python
mi_lista = [1, 2, 3, 2]
mi_lista.remove(2)
print(mi_lista)  # Salida: [1, 3, 2]
```

## 3️⃣ clear()
---
El método clear() elimina todos los elementos de una lista, dejando la lista vacía. Este método es útil cuando queremos vaciar una lista sin tener que reasignarla a una nueva variable.

```python
mi_lista = [1, 2, 3, 4]
mi_lista.clear()
print(mi_lista)  # Salida: []
```

Estos métodos nos permiten realizar una gran variedad de operaciones sobre listas, facilitando tareas como agregar, eliminar, ordenar, y manipular sus elementos.

Cada método tiene su propio propósito y uso adecuado en diferentes escenarios de programación.
